# 筛选排序系统 - 移动端原型

## 项目简介

这是一个专为移动端设计的商品/订单/任务单通用筛选排序系统,适用于奢侈品回收管理场景。

## 功能特性

### 📱 核心功能

1. **排序功能**
   - 10种排序方式:最新/最早创建、最新/最早完成、最新/最早更新、回收价/销售价升降序
   - 全屏下拉展示,点击后同步更新排序按钮文字
   - 黑色30%透明蒙层

2. **筛选功能**
   - 26个筛选维度,支持多选/单选
   - 左侧导航+右侧内容,滚动联动
   - 实时统计已选数量,红点提示

3. **品牌筛选**
   - 推荐品牌快速选择
   - 全部品牌按A-Z字母分组
   - 推荐品牌和全部品牌选中状态双向同步
   - 已选品牌显示在标题行

4. **成员选择**
   - 客服/鉴定师/报价师支持多选
   - 分组展示,支持搜索
   - 二级页面选择,确定按钮确认
   - 已选成员显示在标题行

5. **时间筛选**
   - 快捷时间选项(今天/近7天/近30天/近3个月)
   - 自动填充日期范围
   - 支持自定义日期区间

### 🎨 交互特点

- **单选支持取消** - 所有单选项可点击取消
- **智能联动** - 左右区域实时同步滚动
- **平滑动画** - 所有弹出、滚动都有流畅过渡
- **键盘适配** - 自动处理输入框键盘弹起
- **防误触** - 蒙层点击关闭,内容区点击不冒泡

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 品牌 | 多选 | 推荐品牌+全部品牌,A-Z分组 |
| 品类 | 多选 | 手袋、单肩包、斜挎包、钱包等 |
| 成色 | 多选 | 99新、95新、90新等 |
| 商品状态 | 多选 | 在库、在售、锁定、售出 |
| 仓库 | 单选 | 总部仓、华东仓等,二级页选择 |
| 创建时间 | 单选 | 快捷选项+自定义日期范围 |
| 预约时间 | 日期 | 自定义日期范围 |
| 首次跟进 | 多选 | <30分钟、<1小时、1-2小时等 |
| 跟进间隔 | 多选 | <6小时、6-24小时等 |
| 线索来源 | 多选 | 大众点评、小红书、抖音等 |
| 创建渠道 | 多选 | 自动创建、运营创建、批量导入 |
| 线索类型 | 单选 | 鉴定咨询、报价咨询、回收 |
| 咨询商品数 | 多选 | 1个、2个、3-5个、>5个 |
| 任务优先级 | 单选 | 紧急、正常 |
| 跟进客服 | 多选 | 分组选择,二级页面 |
| 回收方式 | 单选 | 上门回收、到店回收、邮寄回收 |
| 鉴定师 | 多选 | 分组选择,二级页面 |
| 鉴定方式 | 单选 | 总部图片鉴定、上门鉴定 |
| 报价师 | 多选 | 分组选择,二级页面 |
| 报价方式 | 单选 | 总部图片报价、上门报价 |
| 入库时间 | 日期 | 自定义日期范围 |
| 出库时间 | 日期 | 自定义日期范围 |
| 盘点准确率 | 多选 | 100%、99%、95-98%等 |
| 挂签 | 单选 | 已挂签、未挂签 |
| 用户级别 | 单选 | VIP会员、普通会员、新用户 |
| 回收商品数 | 多选 | 无、1件、2件、3-5件等 |

## 技术栈

- 纯HTML + CSS + JavaScript
- 无依赖,开箱即用
- 响应式设计,适配各种移动设备
- 支持iOS和Android

## 快速开始

### 本地预览

1. 下载 `index.html` 文件
2. 用浏览器打开即可
3. 建议使用Chrome开发者工具的移动设备模式

### GitHub Pages 部署

1. 将文件推送到仓库
2. 进入仓库 Settings
3. 找到 Pages 选项
4. Source 选择 main 分支
5. 保存后等待部署完成
